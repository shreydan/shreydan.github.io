## React boilerplate using NPM

- styled-components
- axios
- react-router-dom
- uuid

### Steps:

- `git clone https://github.com/shreydan/react-boilerplate.git <project-name>`
 
  OR
  
  from Github click on `Use this template` and clone the new repository
- change `name` in `package.json`
- run `npm install`
